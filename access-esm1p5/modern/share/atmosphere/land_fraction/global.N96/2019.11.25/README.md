### Links/relationships

This data contains gridded land fractions for ESM1.5 and ESM1.6, which is used in several other ancillary data sets. For example the land fraction data is used to identify ocean points when generating the AMIP SSTs and sea ice files.

## Methodological information

This file is copied from `/g/data/access/projects/access/data/ancil/access_v2`. Please note that this file is not directly used by the model, which instead uses identical data stored in the atmospheric restart files. These fields are the same in both ESM1.5 and ESM1.6.
